---
title: Working with Camera Heading Vectors
---

import { Steps } from "@astrojs/starlight/components";

Use `pure.camera.getHeading()` to obtain a compass heading and convert it into a unit direction vector
suitable for movement, raycasting, or trigonometric functions.

:::note

`getHeading()` returns degrees clockwise from North.
Trigonometric functions expect counter-clockwise from East,
so the angle must be converted before use.

:::

## Steps

<Steps>

1.  Fetch the camera's heading

    ```lua
    local heading = pure.camera.getHeading()
    ```

2.  Convert the heading to radians

    ```lua
    local radians = math.rad(90 - heading)
    ```

    :::note

    `% 360` normalizes the angle to the range `0`-`360`.
    This is optional; `sin` and `cos` work correctly with angles outside that range.

    :::

3.  Compute the direction vector

    ```lua
    local direction = {
      x = math.cos(radians),
      y = math.sin(radians)
    }
    ```

    `x` and `y` form a unit direction vector for the camera's heading.

</Steps>
