title: Working with Camera Heading Vectors
---

Use `pure.camera.getHeading()` to obtain a compass heading and convert it into a unit direction vector
suitable for movement, raycasting, or trigonometric functions.

## Problem

`getHeading()` returns an angle measured clockwise from North.
Trigonometric functions such as `sin` and `cos` expect angles measured counter-clockwise from the positive X-axis (East).
The angle must be converted before use.

## Steps

<Steps>

1.  Fetch the camera's heading

    ```lua
    local heading = pure.camera.getHeading()
    ```

2.  Convert to a mathematical angle

    ```lua
    local mathAngle = 90 - heading
    ```

    :::note

    `% 360` normalizes the angle to the range `0`-`360`.  
    This is optional; `sin` and `cos` work correctly with angles outside that range.

    :::

3.  Convert to radians

    ```lua
    local radians = math.rad(mathAngle)
    ```

4.  Compute the direction vector

    ```lua
    local direction = {
      x = math.cos(radians),
      y = math.sin(radians)
    }
    ```

    `x` and `y` form a unit direction vector for the camera's heading.

</Steps>

## Example

```lua title=".lua"
local heading = pure.camera.getHeading()
local mathAngle = (90 - heading)
local r = math.rad(mathAngle)

local dir = {
  x = math.cos(r),
  y = math.sin(r)
}

print("Camera Heading:", heading)
print("Direction Vector:", dir.x, dir.y)
```

:::tip
The `dir` vector is normalized.
:::
