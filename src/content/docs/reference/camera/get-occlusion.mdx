Returns the ambient occlusion factor at the camera's current world position.

```lua
pure.camera.getOcclusion()
```

:::note

_Vertex Ambient Occlusion_ (VAO) stores a precomputed ambient lighting factor per vertex.
These values approximate how exposed a surface is to ambient light.
Lower exposure occurs in creases, tunnels, and areas enclosed by static geometry.

:::

## Description

Returns the current occlusion level affecting the camera.
This is derived from the VAO patch data at the camera's current world coordinates.
