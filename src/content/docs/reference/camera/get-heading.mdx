---
title: getHeading
description: Returns the compass heading of the camera in degrees.
---

Get the heading of the camera.

```lua
pure.camera.getHeading()
```

Returns the camera heading as a `number` in degrees, in the range `0`-`360`.
`0°` corresponds to North and increases clockwise.

To use this value with trigonometric functions, see [Working with Camera Heading Vectors](/guides/working-with-camera-heading-vectors).
