## Returns

| Type     | Description                                                                       |
| :------- | :-------------------------------------------------------------------------------- |
| `number` | The multiplier factor used to balance daylight against artificial track lighting. |

## Description

`getSunLampDifference` returns a ratio used to maintain visual balance
between natural environmental light (the sun) and artificial light sources (track lamps).

The value is calculated based on the current sun elevation and the `light.daylight_multiplier` configuration setting.
