```lua
pure.light.getMoon()
```

## Returns

| Type  | Description                  |
| :---- | :--------------------------- |
| `rgb` | The current moonlight color. |

## Description

`getMoon` returns the color of the moonlight currently calculated by the weather engine.

Pure generates an exclusive instance of this `rgb` variable for the calling script.
This allows the object to be manipulated within the current frame without affecting the internal engine state or other scripts.
The instance is reset at the start of the next frame.
