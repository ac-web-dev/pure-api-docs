## Parameters

| Parameter | Type      | Description                                                                    |
| :-------- | :-------- | :----------------------------------------------------------------------------- |
| `x`       | `number?` | The global light multiplier. If `nil` or omitted, the value defaults to `1.0`. |

## Description

`setGlobalBrightness` sets a multiplier that affects all environmental light sources simultaneously,
including the sun, moon, and ambient light.

The value scales the intensity of the world lighting before it is processed by the post-processing pipeline.
It is used to adjust the overall exposure of the scene without modifying individual light source components or multipliers.
