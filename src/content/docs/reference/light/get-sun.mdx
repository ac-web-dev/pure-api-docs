```lua
pure.light.getSun()
```

## Returns

| Type  | Description                 |
| :---- | :-------------------------- |
| `rgb` | The current sunlight color. |

## Description

`getSun` returns the color of the sunlight currently calculated by the weather engine.

Pure generated an exclusive instance of this `rgb` variable for the calling script.
This allows the object to be manipulated within the current frame without affecting the internal state or other scripts.
The instance is reset at the start of the next frame.
