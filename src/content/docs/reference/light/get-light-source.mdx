## Returns

| Type  | Description                             |
| :---- | :-------------------------------------- |
| `rgb` | The current primary light source color. |

## Description

`getLightSource` returns the color of the dominant light source in the scene.
This value dynamically transitions between the calculated sun light and moonlight based on their respective elevations and multipliers.

Pure generates an exclusive instance of this `rgb` variable for the calling script.
This allows the object to be manipulated within the current frame without affecting the internal engine state or other scripts.
The instance is reset at the start of the next frame.
