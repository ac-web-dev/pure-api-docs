## Parameters

| Parameter | Type      | Description                                                                                                                        |
| :-------- | :-------- | :--------------------------------------------------------------------------------------------------------------------------------- |
| `x`       | `number?` | The base gamma value. A value less than zero sets the internal state to `-1`. If the value is `nil` or omitted, defaults to `1.0`. |

## Description

{/* see:

    pure_sharing.lua,
    world_sun_light.lua

*/}

`setLambertGamma` sets the base gamma value used to calculate light distribution across material surfaces.
This value influences the `ksDiffuse` component of materials based on their angle relative to the light source.

Pure stores the value internally and, when `ac.setLambertGamma` is available,
applies its reciprocal to the rendering engine.
This calculation accounts for internal multipliers and sun-angle dependencies unless those adaptations are explicitly disabled.
