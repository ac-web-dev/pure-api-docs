This site is a documentation project for Pure, a weather and lighting system for Assetto Corsa with CSP.
It aims to provide a structured technical reference for the Lua API used to develop weather and post-processing scripts,
along with guides and resources that follow the [Diátaxis framework](https://diataxis.fr/).

Pure is developed by Peter Boese. Support the development and access releases via his [Patreon](https://www.patreon.com/peterboese).

:::note
Content is versioned to match the releases of Pure, with the latest being `3.21` as of May 4th, 2026.
:::

Due to the nature of the project, contributions to the documentation are welcome and encouraged, see the [contributing guide](/contributing) for more details.

## Legal

Pure is a weather and graphics modification for CSP, developed and owned by Peter Boese.

This documentation is a resource for the API provided by Pure, and does not distribute Pure software, binaries, or protected assets.

## Requirements

To use Pure, and the scripting interface documented here, the following dependencies must be met:

- [Assetto Corsa](https://store.steampowered.com/app/244210/Assetto_Corsa/)
- [Custom Shaders Patch](https://acstuff.club/patch/)
- [Pure](https://www.patreon.com/peterboese)
